Willkommen beim VeKa-Center-Auskunftsdienst.

Die folgenden REST-Operationen stehen derzeit zur Verfügung:

  • Grundversicherung für eine Karte/Person prüfen: http://localhost:8070/vekaapi/checkbaseinsurance?firstName=VORNAME&officialName=NACHNAME&dateOfBirth=TT.MM.JJJJ&baseInsuranceNumber=NUMMER
  • Versicherer zu einer Kartennummer erhalten: http://localhost:8070/vekaapi/getinsurerofcard?baseInsuranceNumber=NUMMER
  • (Neuen) Versicherer hinzufügen: http://localhost:8070/vekaapi/addinsurer?name=NAME&street=STRASSE&housenumber=HAUSNUMMER&plz=POSTLEITZAHL&town=ORT

Zudem können Sie hier eine Liste aller aktuell registrierten Versicherern abrufen.
